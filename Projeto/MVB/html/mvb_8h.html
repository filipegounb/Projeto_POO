Esta função altera o status (livre ou ocupado) de um registro no arquivo desejado.

Parâmetros:
<opcao>- Recebe o nivel do usuario ('1' para adicionar Master; '2', para Professor; '3', para Aluno)
<posicao>- Recebe o valor que indica a posição relativa do registro à ser alterado
<novo_status>- Recebe o valor do novo status do registro ('0', para livre; '1', para ocupado)
Valores Retornados:
<0>- alteracao feita com sucesso
<-1>- indica que o novo valor que se quer atribuir ao registro é inválido (diferente de '0' e de '1')
<-2>- indica que o arquivo que se deseja alterar não existe
<-3>- indica que o registro que deve ser alterado não existe, ou seja, o tamanho do arquivo é menor do que a posição do registro que se quer alterar
<-4>- indica que o valor passado para o primeiro argumento é inválido

Parâmetros:
<opcao>- Recebe o nivel do usuario ('1' para adicionar Master; '2', para Professor; '3', para Aluno)
Valores Retornados:
<Valor_nao_negativo>- a posicao em que o registro foi incluido
<-1>- indica que o registro foi inserido no final do arquivo ou que o arquivo não existia e agora foi criado e preenchido com o novo dado
<-2>- indica que o valor passado como parâmetro para a função é inválido
