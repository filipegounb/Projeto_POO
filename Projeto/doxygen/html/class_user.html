Descrição detalhada

Esta classe checa a existencia de usuarios e solicita a exlusao de usuarios.

bool User::existe_user ( const string  nome,
const string  matricula,
const string  senha,
const char *  nome_arquivo = "dados/Master.txt" 
)

Este metodo solicita a checagem da existencia de um usuário Master com os dados recebidos.

Parâmetros:
<nome>- Nome do usuario
<matricula>- Matricula do usuario
<senha>- Senha do usuario
<*nome_arquivo>- Arquivo que armazena os dados dos usuarios Master cadastrados (por padrao, tem o valor "dados/Master.txt")
Retorna:
- Retorna 'true' se o usuario existe; 'false', caso contrario
