int Recupera_Notas::obter_conjunto_notas ( const string &  matricula,
const string &  disciplina_professor,
const char *  nome_arquivo = "dados/Notas.txt" 
)

Este metodo le as notas de um aluno e as armazena em um vetor.

Parâmetros:
<matricula>- Matricula do Aluno
<disciplina_professor>- Disciplina ministrada pelo Professor (para desativar o filtro de busca com disciplina, deve-se entrar com o valor " " - espaco - nestes campo)
<*nome_arquivo>- Nome do arquivo que possui as notas (por padrao, tem o valor "dados/Notas.txt")
Retorna:
- Retorna a quantidade de disciplinas que tem notas publicadas para esta matricula
Nota Recupera_Notas::obter_uma_nota ( )

Este metodo retorna uma nota, sendo que sempre retorna a nota mais recente ainda presente no vetor 'conjunto_notas'.

Parâmetros:
-Nao possui parametros de entrada
Retorna:
- Retorna uma nota (disciplina e as quatro notas desta disciplina, em um objeto do tipo Nota) na ordem inversa do vetor 'conjunto_notas'
